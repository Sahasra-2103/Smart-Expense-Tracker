## 🚀 Get Started in 3 Steps

### Step 1: Install Backend (2 minutes)
```bash
cd backend
npm install
```

### Step 2: Generate Frontend (10 seconds)
```bash
npm run setup
```

### Step 3: Install Frontend (2 minutes)
```bash
cd ../frontend
npm install
```

---

## ▶️ Run the Application

### Option A: Automatic (Recommended)

**Windows:**
```bash
quickstart.bat
```

**macOS/Linux:**
```bash
bash quickstart.sh
```

### Option B: Manual

**Terminal 1 - Backend:**
```bash
cd backend
npm run dev
```

**Terminal 2 - Frontend:**
```bash
cd frontend
npm start
```

## ⚙️ Before Starting

### Make Sure You Have:
- [ ] Node.js v14+ installed
- [ ] npm installed
- [ ] MongoDB running (local or cloud connection string)

### Create Configuration:
```bash
cd backend
cp .env.example .env
# Edit .env with your:
# - MONGODB_URI
# - JWT_SECRET (any random string)
# - GEMINI_API_KEY (optional for AI features)
```

## 🆘 Quick Help

| Issue | Fix |
|-------|-----|
| `npm: not found` | Install Node.js from nodejs.org |
| `port already in use` | Kill process or change port in .env |
| `MongoDB connection error` | Check MONGODB_URI in .env |
| `npm run setup not found` | Make sure you're in backend folder |
